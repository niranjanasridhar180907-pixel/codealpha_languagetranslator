# 🌍 AI-Powered Language Translation Tool

## 📌 Project Overview

The AI-Powered Language Translation Tool is a web application developed using Python and Streamlit that enables users to translate text between multiple languages instantly. The application provides an interactive user interface and additional features such as automatic language detection, text-to-speech conversion, translation history, and downloading translated text.

This project was developed as part of the **CodeAlpha Artificial Intelligence Internship**.

---

## 🚀 Features

* 🌐 Translate text between multiple languages
* 🔍 Automatic language detection
* 🔊 Text-to-Speech (TTS) for translated text
* 📜 Translation history tracking
* 📥 Download translated text as a file
* 📊 Character count display
* 📋 Copy translated text
* 🎨 User-friendly Streamlit interface

## ⚙️ Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/niranjanasridhar180907-pixcel/CodeAlpha_LanguageTranslator.git
cd CodeAlpha_LanguageTranslator
```

### Step 2: Install Required Libraries

```bash
pip install -r requirements.txt
```

### Step 3: Run the Application

```bash
streamlit run app.py
```
